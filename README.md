## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
[Nest GraphQL](https://docs.nestjs.com/graphql/quick-start)

- Rest API만 만들어온 나에게는 조금 낯설은 방식이었습니다.
- 하지만, NestJS는 GraphQL을 매우 훌륭하게 지원합니다.
- 사용하는 방식도 기존 Swagger 및 Controller를 만드는 방식과 많이 다르지 않습니다.
- 아직까지는 GraphQL을 사용하는 회사나 프로젝트를 진행한적이 없어서 알아두고 익히려 합니다.
